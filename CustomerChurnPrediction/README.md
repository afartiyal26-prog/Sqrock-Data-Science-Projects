# Customer Churn Prediction

## Overview
This project predicts whether a customer is likely to leave (churn) or stay using Machine Learning. It analyzes customer information such as age, monthly bill, and contract duration to make predictions.

## Features
- Data Loading
- Data Analysis
- Missing Value Check
- Statistical Summary
- Data Visualization
- Logistic Regression Model
- Customer Churn Prediction
- Model Evaluation using Accuracy Score and Confusion Matrix

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
