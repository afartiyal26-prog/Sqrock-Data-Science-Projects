# Sales Forecasting System

## Overview
This project predicts future sales using Machine Learning. It analyzes historical sales data, performs data visualization, and uses a Linear Regression model to forecast future sales.

## Features
- Data Loading
- Data Cleaning
- Missing Value Check
- Sales Trend Analysis
- Professional Data Visualization
- Linear Regression Model
- Future Sales Prediction
- Model Evaluation using MAE and RMSE

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Visualizations
- Sales Trend Line Chart
- Monthly Sales Bar Chart
- Sales Distribution Pie Chart

- Box Plot
- Moving Average Trend
- Actual vs Predicted Sales
- Future Sales Forecast

## Machine Learning Model
- Linear Regression

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## Future Prediction
The model predicts sales for the next six months based on historical data.
