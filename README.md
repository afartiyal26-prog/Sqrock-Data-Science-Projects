# Sqrock Data Science Projects

This repository contains three Data Science internship projects completed as part of the Sqrock IT Solutions Internship.

## Projects

1. Sales Forecasting System
Predicts future sales using Machine Learning.

2. Student Performance Predictor
Predicts students' final scores using Linear Regression.

3. Customer Churn Prediction
Predicts whether a customer is likely to churn using Logistic Regression.

##Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
