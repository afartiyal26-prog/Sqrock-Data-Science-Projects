# Student Performance Predictor

## Overview
This project predicts students' final scores using Machine Learning. It analyzes student study data, attendance, and previous scores to estimate future academic performance.

## Features
- Data Loading
- Data Cleaning
- Missing Value Analysis
- Statistical Summary
- Professional Data Visualization
- Linear Regression Model
- Student Score Prediction
- Model Evaluation using MAE and RMSE

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Visualizations
- Final Score Distribution
- Hours Studied vs Final Score
- Attendance vs Final Score
- Actual vs Predicted Scores
- Future Student Score Prediction

## Machine Learning Model
- Linear Regression

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## Future Prediction
The model predicts the final scores of new students based on study hours, attendance, and previous scores.
